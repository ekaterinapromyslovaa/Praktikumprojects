# Анализ финансирования стартапов

## Описание проекта

Финансовая компания, работающая с венчурными инвестициями, хочет понять закономерности финансирования стартапов и оценить перспективы выхода на рынок с покупкой и развитием компаний. Для этого необходимо провести исследование на исторических данных.

В проекте проведена работа с данными:
- о компаниях
- об объёмах и типах привлечённых инвестиций
- о возвратах средств (дополнительная статистика)

---

## Цели и задачи проекта

### Цель
Предобработать датасет, убедиться, что данные не излишни и соответствуют здравому смыслу, а также ответить на вопросы заказчика о предобработке и значении данных для бизнеса. Выполнить анализ на основе обработанных данных и получить выводы для принятия решений о выходе на инвестиционный рынок в сфере стартапов.

### Задачи

1. Познакомиться с данными, провести предобработку без потери важных данных и искажения результатов
2. Идентифицировать столбцы для объединения данных из разных таблиц
3. Сделать выводы о доверии к данным о сотрудниках стартапов и их образовании
4. Разобраться, что означают покупки за 0 или 1 доллар
5. Определить средние цены в зависимости от категории стартапов и количества раундов финансирования перед покупкой
6. Рассчитать и отобразить численные показатели и динамику
