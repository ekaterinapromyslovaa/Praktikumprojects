# Учебные проекты — Аналитик данных / Продуктовый аналитик

## О репозитории

В этом репозитории хранятся мои Python-проекты, созданные в ходе обучения в "Яндекс.Практикуме" по направлениям «Аналитик данных» и «Продуктовый аналитик»

Каждый проект представляет собой самостоятельное исследование, включающее:
- предобработку и анализ данных
- расчёты ключевых метрик
- визуализации
- выводы и рекомендации

---

## Проекты

### 1. Анализ маркетплейса

**Суть проекта:**  
Оценка «здоровья» быстрорастущего маркетплейса. Рассчитаны ключевые метрики (выручка, ARPU, ARPPU, LTV, CAC, ROI, Retention), проведён анализ воронки, исследованы каналы привлечения и товарные категории. Сформулированы гипотезы для роста. Проведен А/В-тест маркетинговой кампании по данным о команде

**Проведенные исследования:**
- Верхнеуровневая оценка продуктовых и финансовых метрик
- Точечная аналитика по разрезам (каналы, категории, сегменты)
- A/B-тестирование маркетинговой кампании (бонус на первую покупку для пользователей, привлеченных через TikTok)

**Иcпользуемые библиотеки:** pandas, matplotlib, seaborn, scipy, statsmodels

**Статус:** Завершён

---

### 2. Анализ финансирования стартапов

**Суть проекта:**  
Исследование исторических данных о финансировании стартапов. Проведена предобработка, анализ выбросов, изучены необычные сделки за 0-1$, определены средние цены в зависимости от категории стартапов и количества раундов финансирования перед их покупкой.

**Иcпользуемые библиотеки:** pandas, matplotlib, seaborn, Phi_K

**Статус:** Завершён

---

### 3. Анализ сервиса доставки еды

**Суть проекта:**  
Исследование поведения пользователей и сегментация ресторанов-партнеров и формирование рекомендаций для повышения удержания пользователей и улучшения опыта взаимодействия с партнерами.

**Методы, использованные в анализе:**
- RFM-сегментация пользователей
- Кластеризация ресторанов методом K-Means

**Иcпользуемые библиотеки:** pandas, matplotlib, sklearn.cluster (KMeans)

**Статус:** Завершён

## Как работать с репозиторием

1. Перейдите в папку интересующего проекта
2. Откройте `README.md` внутри папки проекта — там описаны цели, задачи и результаты
3. Запустите Jupyter Notebook (`.ipynb`) для просмотра проекта
